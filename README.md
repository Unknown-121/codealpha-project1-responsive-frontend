Northbridge Consulting — Responsive Frontend Interface

Project 1 of the CodeAlpha Full Stack Development internship: a fully responsive frontend interface built with pure HTML, CSS, and vanilla JavaScript, no frameworks.

Live demo: just open index.html in any browser, no install needed.

About

This is a landing page for a fictional consulting firm, built to demonstrate mobile first responsive design, semantic HTML, and interactive form validation without relying on any frontend framework or library.

Features

Fully responsive: mobile first layout that adapts at 768px (tablet) and 1024px (desktop) breakpoints
Semantic HTML5: header, nav, main, section, footer for accessibility and SEO
CSS Grid and Flexbox: Grid for page level layout, Flexbox for components like the nav and cards
Fluid typography: using clamp() so text scales smoothly across screen sizes
Working consultation booking form: real client side JavaScript validation (email format, phone format, date checks, required fields) with inline error messages and a live confirmation list
Accessible: visible focus states, aria live status updates, keyboard friendly mobile menu

Tech stack

HTML5
CSS3 (Grid, Flexbox, custom properties, clamp())
Vanilla JavaScript (DOM manipulation, form validation, no libraries)
Google Fonts (Montserrat, Roboto)

Design

Color palette: Mocha Mousse #A5856F, Ethereal Blue #A0D4E0, Moonlit Grey #F2F0EA
Typography: Montserrat (headings), Roboto (body)
